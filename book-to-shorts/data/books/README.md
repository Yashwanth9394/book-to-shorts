# Books Directory

Place your PDF books here to be processed.

## How to Add a Book

1. Copy your PDF file to this directory:
   ```bash
   cp /path/to/your/book.pdf data/books/
   ```

2. Run the analysis:
   ```bash
   npm run analyze
   ```

## Supported Formats

- PDF files (.pdf)
- Non-encrypted PDFs only
- Maximum size: 32MB (recommended)

## Best Books to Process

Books with clear, dramatic stories work best:
- History books with specific events
- Business books with case studies
- Biographies with interesting anecdotes
- Philosophy with concrete examples

## What to Avoid

- Dense technical manuals
- Books with mostly abstract theory
- Heavily encrypted or protected PDFs
- Scanned PDFs with poor OCR

## Examples

Good choices:
- "The 48 Laws of Power" by Robert Greene
- "Sapiens" by Yuval Noah Harari
- "The Art of War" by Sun Tzu
- "Meditations" by Marcus Aurelius
- "Thinking, Fast and Slow" by Daniel Kahneman

## Tips

- Start with shorter books (100-200 pages) for testing
- Make sure PDF text is selectable (not just images)
- Public domain books work great for testing
